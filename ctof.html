Celcius To Farenngeit

Bob is building an iOT device acting as a sensor. Given a temperature in celsius C, he request you to write a program to convert celsius and return farenheit value.
Round off the Result to the lowest possible Integer Value.

use Formula F = 1.8 C + 32

Input:
The testcase contains an Integer C denoting celsius value.

Output:
For the testcase, output the temperature in farenheit, rounded off to the lowest possible Integer value

Example 1:

Input:
51

Output:
123

Example 1:

Input:
65

Output:
149
